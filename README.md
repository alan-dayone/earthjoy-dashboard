### Getting started

#### Prerequisite
- Node.js LTS version
- Existing API backend

#### Run on localhost
Step 1: Copy `example.env` to a new file name `.env`, change host and DB information if you want.
Step 2: Run server
```bash
npm install
npm run dev
```
